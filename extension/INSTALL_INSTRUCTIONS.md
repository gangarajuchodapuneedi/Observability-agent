# Extension Installed in Main Workspace

The extension has been copied to your Cursor extensions folder:
`C:\Users\SHAIK\.cursor\extensions\observability-agent-0.0.1`

## To Activate the Extension:

1. **Reload Cursor/VS Code:**
   - Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
   - Type: "Developer: Reload Window"
   - Press Enter

2. **Or restart Cursor completely**

3. **After reload, use the extension:**
   - Press `Ctrl+Shift+P`
   - Type: "Observability Agent: Open Chat"
   - Press Enter

## Verify Installation:

- Go to Extensions view (`Ctrl+Shift+X`)
- Search for "Observability Agent"
- You should see it listed as installed

## Note:

Make sure the backend API server is running on `http://localhost:8000` before using the extension!
